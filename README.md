#  Swiggy Performance Analytics Dashboard

##  Project Overview
This dashboard analyzes Swiggy’s business performance by evaluating restaurant count, user ratings, revenue trends, and discounts across cities.

---

##  Objectives
- Analyze restaurant distribution by city
- Study user rating behavior
- Track revenue trends year-wise
- Identify preferred payment modes
- Understand discount impact

---

##  Key KPIs
- Total Restaurants
- Total Revenue
- Average Discount Applied
- Most Used Payment Mode

---

##  Tools Used
- Power BI
- Data Visualization
- Data Analysis
- Business Intelligence Concepts

---

##  Dashboard Preview
![Swiggy Dashboard](swiggy_dashboard.png)

---

## Insights
- Mumbai has the highest restaurant count
- Card is the most preferred payment mode
- Revenue shows a decline from 2024 to 2025
- Discounts significantly influence order value
